# Minecraft 知识库来源与维护

本知识包使用原创归纳文字编写，以下来源用于核验版本敏感机制和持续维护：

- Minecraft 官方网站与版本更新日志：https://www.minecraft.net/
- Minecraft Wiki：https://minecraft.wiki/
- Redstone Comparator：https://minecraft.wiki/w/Redstone_Comparator
- Redstone Repeater：https://minecraft.wiki/w/Redstone_Repeater
- Commands：https://minecraft.wiki/w/Commands
- Villager：https://minecraft.wiki/w/Villager
- Brewing：https://minecraft.wiki/w/Brewing

维护原则：

1. 对明确受版本影响的数值、配方、命令和生成规则，记录对应版本。
2. Java 版与基岩版结论分开写，不以其中一个版本替代另一个版本。
3. 不复制受版权保护的长篇 Wiki 内容；以简洁的原创解释和链接指向原始资料。
4. 新增文档应保持一个主题一个文件，并在标题后添加便于检索的中文标签。
