# Minecraft AstrBot 知识库

这是一个面向 AstrBot 检索增强回答的中文 Markdown 知识包。内容按主题拆分，适合导入名为 `Minecraft` 的知识库；它覆盖常用机制与概念，不替代特定版本的完整更新日志。

## 导入建议

1. 在 AstrBot 的知识库管理页面创建知识库，名称填写 `Minecraft`。
2. 将本目录中的 `00` 到 `06` 号 Markdown 文档全部导入；`SOURCES.md` 用于人工核验，可按需要一并导入。
3. 推荐使用约 `500-800` 字符的分块大小和 `80-120` 字符的重叠。这样“比较器”“附魔”“下界要塞”等问题通常能命中完整的小节。
4. 在本插件配置中启用知识库优先检索，并将 `knowledge_base_name` 设置为 `Minecraft`。

## 版本边界

- 基础机制以 Java 版与基岩版共有内容为主。
- 指令、红石时序、刷怪、战斗冷却、村民交易和结构生成存在版本差异时，文档会标注“Java 版”“基岩版”或“需按版本核对”。
- 对版本敏感的配方、概率、结构位置、掉落表和命令参数，应在实际服务器版本中验证，并优先查阅官方更新日志或 Minecraft Wiki。

## 文档清单

- `00_基础与版本边界.md`：游戏循环、模式、时间与难度。
- `01_生存与发展.md`：资源、工具、农业、下界和末地进度。
- `02_红石与自动化.md`：红石强度、元件、比较器、常用电路。
- `03_战斗_附魔_酿造.md`：装备、防御、附魔、药水和首领。
- `04_世界_生物_村民.md`：维度、结构、生物、村民与繁殖。
- `05_指令与技术玩法.md`：常用命令、规则与排错边界。
- `06_常见问答.md`：适合直接检索的短问短答。
- `SOURCES.md`：维护和核验来源。
