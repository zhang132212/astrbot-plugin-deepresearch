# Minecraft 红石与自动化

标签：红石、比较器、中继器、观察者、活塞、自动化、信号强度

## 红石信号基础

红石信号强度范围为 0 到 15。红石粉会随距离衰减，普通红石粉线路每经过一格通常降低 1 级强度；红石中继器可重新输出满强度信号并提供延迟与单向传输。火把可作为反相器：输入有电时火把熄灭，输入无电时火把点亮。

## 中继器

中继器有单向性，常用于延长线路、隔离回流、增加延迟和制作脉冲。右键可调整延迟档位；最短延迟为 1 个红石刻。中继器侧面可被锁定，锁定后保持当前输出状态直到解除锁定。

## 比较器

比较器读取后方主输入 R 和左右两侧中较强的侧向输入 S。

- 比较模式：若 R 大于等于 S，输出 R；否则输出 0。
- 减法模式：输出 `max(R - S, 0)`。
- 输出延迟为 1 个红石刻。

比较器还可读取容器填充度和部分方块状态，例如箱子、漏斗、熔炉、讲台、物品展示框或唱片机。它读取的是方块状态转换出的强度，因此适合库存检测、分类机、音乐机和密码锁。

## 观察者与活塞

观察者在检测到前方方块状态变化时，从背面发出短脉冲。它常用于作物成熟检测、移动方块检测和脉冲触发。活塞可以推动多数方块；粘性活塞还能拉回部分方块。不可移动方块、准连接、更新顺序和版本差异是复杂活塞电路常见问题。

## 常见自动化设计

- 自动熔炉：利用漏斗输入燃料和原料，并用漏斗收集产物。
- 分类系统：依据物品过滤器和红石强度维持固定数量的过滤物品。
- 作物农场：用水流、村民、观察者或活塞触发收集；是否适用取决于作物与版本。
- 库存报警：比较器读取容器强度，当低于或高于阈值时触发灯、音符盒或消息。

## 红石排错顺序

1. 确认线路是否获得足够强度，而非只看红石粉是否发亮。
2. 检查中继器朝向、延迟和是否被锁定。
3. 检查比较器是比较模式还是减法模式。
4. 确认相关区块已加载，且服务器没有因性能限制暂停装置。
5. 对 Java 版与基岩版的复杂电路，不要假设两者更新顺序完全一致。
