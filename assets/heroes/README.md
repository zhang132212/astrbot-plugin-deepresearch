# Random report hero images

Place one or more authorized PNG files in this directory. Every report image will randomly
choose one file from this directory for its top-right hero visual.

Recommended size: 900 x 760 pixels. Keep the subject near the center or right and avoid
important details at the bottom edge because the report header masks that area.

`assets/report_hero.png` remains supported for backward compatibility. When custom PNG files
exist, they take precedence over the built-in CSS visuals.
